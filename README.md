# Boo Backend with Node.js
To start the project, run these commands:

# Installation:
1. git clone https://github.com/your-username/boo-backend.git
2. npm install
Start the application:

3. npm start

# Setup
1. Open the terminal
2. Run npm install
3. Run npm start
4. To run the test cases, use this command:

# Test
1. npm test
